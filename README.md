An Open Source Weather App made with Typescript & ReactJs.
## Features

- Search Weather Conditions Globally
- Temperature Converter
- Dark Mode
- Show Country Info
- Interactive 3D Map of the Earth



## Installation

Install my-project with npm

```bash
  npm install my-project
  cd my-project
```
